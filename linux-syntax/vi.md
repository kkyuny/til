## vi 편집기로 파일 열기
- vi 파일명

## 입력모드로 전환
- i : 현재 커서에서 입력
- a : 다음 커서부터 입력
- o : 다음 줄부터 입력

## 명령모드(esc로 전환한다.)
- :q! : 저장없이 나가기
- :wq! : 저장하고 나가기
- x : 한칸 삭제
- dd : 한줄 삭제
- yy : 한줄 복사
- p : 복사내용 다음줄에 붙여넣기
- 시트프 + g : 마지막 줄로 이동
- gg : 파일의 첫줄로 이동
- 컨트롤 + f : 다음 페이지로 이동
- 컨트롤 + b : 이전 페이지로 이동
- /문자열: 문자열을 검색한다.
