# 리눅스 기본 명령어

## 사용자가 현재 위치해 있는 폴더 경로 출력 명령어
- pwd

## 현재 위치에서 파일 및 폴더 목록 조회
- ls

## 자세히
- ls -l

## 자세히 + 숨김파일
- ls -al

## 자세히 + 숨김파일 + 시간순 정렬
- ls -alt(오름차순), ls -alrt(내림차순, reverse time?인듯)

## 디렉토리 생성
- mkdir 디렉토리이름

## 경로 이동
- cd 경로이름, cd ..(상위 폴더로 이동), cd /(루트 디렉터리로 이동), cd -(직전에 있었던 폴더로 이동), cd(자신의 홈 경로로 이동)

## 파일 생성
- touch 파일이름

## 파일 내용 확인
- cat 파일이름

## 명령어 취소
- 컨트롤 + c

## 상위/하위 n줄 내용 확인
- head -n 파일이름, tail -n 파일이름

## 실시간으로 추가되는 마지막 내용 확인
- tail -f 파일이름

## 기록을 남기는 명령어(콘솔,파일)
- echo "내용": 콘솔에 내용 출력
- echo "내용" > 파일이름: 파일에 내용 저장(덮어쓰기)
- echo "내용" >> 파일이름: 파일에 내용 저장(추가)

## 히스토리 조회
- history

## 파일 삭제
- rm 파일이름

## 강제 삭제
- rm -f 파일이름

## 디렉토리 삭제
- rm -r -rf 디렉토리이름(디렉토리 하위의 디렉토리 + 파일이 모두 삭제 된다.)

## 파일 복사
- cp 복사할파일이름 복사될파일이름

## 디렉토리 복사
- cp -r 복사할디렉토리이름 복사될경로

## 파일 이동(잘라내서 붙여넣기)
- mv 파일이름 이동할경로(상대,절대): 파일이 입력한 경로로 이동된다.
- mv 파일이름 새로운파일이름: 파일 이름이 변경된다.

## 내용이나 특정패턴 찾기(파일 내에 있는 문자열 찾기)
- grep 찾을내용 파일이름
- grep -r 찾을내용 디렉토리경로: 디렉토리는 리컬시브하게 찾아아야한다.
- grep -ri 찾을내용 디렉토리경로: 대소문자 구분 없이 검색

## 파일, 디렉토리 찾기
- find 디렉토리경로 -name 찾을이름
- find 디렉토리경로 -name 파일이름 type -f: 타입을 명시해주면 해당 타입만 찾는다.

## find + grep
- find 디렉토리경로 -name 찾을이름 | xargs grep -rni 찾을내용: 찾은 경로 내에서 내용을 찾는다.

## find + exec
- find 디렉토리경로 -name 찾을이름 -exec grep -rni 찾을내용 {} \;: {}에 찾은 대상들이 들어간다. \;은 구문의 끝을 의미
- find 디렉토리경로 -name 찾을이름 -exec cp {} 디렉토리이름 \;: {}에 찾은 대상들이 복사된다.
- find 디렉토리경로 -name 찾을이름 -exec echo {} \;: {}에 찾은 대상들이 출력된다.
